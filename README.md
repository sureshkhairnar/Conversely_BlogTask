# Conversely_BlogTask
# Full-Stack Project

## Description

This project includes a Node.js backend API and a React frontend application. The backend handles API requests and data management, while the frontend provides a user interface for interacting with the backend services.

## Project Structure

# blogserver (backend)

### Prerequisites

- Node.js
- npm or yarn
- MongoDB
- Express

# Install dependencies

- npm install

# Run server

- npm start

# BlogClient (frontend)

### Prerequisites

- React.js
- npm or yarn
- MUI

# Install dependencies

- npm install

# Run client app

- npm start
